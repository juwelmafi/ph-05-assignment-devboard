Question-1: What are the different ways to select an element in the DOM?

We can select a DOM element using getElementById(), getElementsByClassName(), getElementsByTagName(), querySelector(), and querySelectorAll(), depending on ID, class, or tag.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

The innerHTML property returns or sets HTML content inside an element, with tags. And innerText shows only visible text, ignores the tag. The textContent returns all text within an element, including hidden content, without formatting.

Question-3: What is event delegation in the DOM?

Event delegation is a technique in the DOM where a parent element handles events for its child elements using event bubbling. Instead of adding event listeners to multiple child elements, we attach a single listener. When an event occurs on a child, it bubbles up to the parent.

Question-4: What is event bubbling in the DOM?

Event bubbling in the DOM is a process where an event starts from the target element and propagates up through its parent elements. When an event occurs on a child element, it first apply on that element, then bubbles up.

Question-5: How do you create, add, and remove elements using JavaScript?

To create an element, we use document.createElement(), it generates a new HTML element. To add it to the DOM, we use appendChild() or append() to insert it at the end of a parent element. If we need to remove an element, we use remove(), or removeChild().
